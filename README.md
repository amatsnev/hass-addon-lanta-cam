# Lanta Camera Proxy — Home Assistant Add-on

Прокси-сервер для камер видеонаблюдения провайдера **ЛанТа** (cam.lanta-net.ru) с автоматическим обновлением токенов.

## Возможности

- Автоматическая авторизация и обновление токенов каждые 1.5 часа
- Проксирование HLS/fMP4 видеопотоков
- Конвертация fMP4 → MPEG-TS через ffmpeg для совместимости с go2rtc
- REST API для списка камер и превью

## Установка

Подробная инструкция: [INSTALL.md](INSTALL.md)

## Камеры

Домофоны, внутренние и лифтовые камеры — автоматически определяются после авторизации.

## Endpoints

| Путь | Описание |
|------|----------|
| `/cameras` | JSON-список камер с URL-потоков |
| `/stream/{cam_id}/{path}` | Проксирование HLS/fMP4 потока |
| `/ts/{cam_id}` | MPEG-TS поток (через ffmpeg, для go2rtc) |
| `/preview/{cam_id}` | JPEG-превью камеры |
| `/refresh` | Принудительное обновление токенов (POST) |
